Setting Up GitHub

In this page you will find neccessary steps to create a GitHub account, create a repository and a web page.

  1. Create GitHub account
  2. Create a repository and a webpage

Create GitHub account

Visit GitHub page and click in Sign Up

Type your email in input

Type your password in input

Type a unique user in input

Type y if you want to receive updates and announcements and n if not

Verify your account following the intructions of captcha

Finally type the code that was sent to your email

Now that you have an account you can sign in GitHub login page

Create GitHub repository and a webpage

Go to profile options and click in Your repositories option

Then click in New button

Set up your repository's name typing the name in input

After successful creation of the repository you can upload a file from your computer o create a file directly in the browser

If you create a file in browser type file name in input and add index content

Add a descriptive message and click in commit

After commit you can type "githubusername.github.io" to access created webpage Created page
